# Odin Recipes

A simple recipe website project built with HTML as part of The Odin Project curriculum.

## Description

This project demonstrates basic HTML skills including:
- HTML document structure
- Links and navigation
- Images
- Lists (ordered and unordered)
- Proper semantic HTML elements

## Recipes Included

- **Meat Loaf** - A classic comfort food dish
- **Zucchini Rollatini** - Zucchini ribbons filled with ricotta and herbs
- **Mushroom Stir Fry** - A vegetarian stir-fry with various mushroom varieties

## Project Structure

```
odin-recipes/
├── index.html
├── recipes/
│   ├── meat-loaf.html
│   ├── zucchini-rollatini.html
│   └── mushroom-stir-fry.html
└── README.md
```

## How to View

Simply open `index.html` in your web browser to view the recipe collection.

## Skills Demonstrated

- Creating and structuring HTML pages
- Implementing navigation between pages
- Using semantic HTML elements
- Organizing content with headings and lists
- Embedding images from external sources
